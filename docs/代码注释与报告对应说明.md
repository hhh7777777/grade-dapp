# 代码注释与课程设计报告对应说明

本文档用于辅助撰写《基于以太坊的链上学生成绩管理 DApp 开发》课程设计报告，可将源码中的中文注释与报告章节对应起来。

## 1. 环境搭建

- `hardhat.config.js`：说明 Solidity 编译器、本地 Hardhat 网络、localhost 网络和项目目录结构。
- `frontend/vite.config.js`：说明前端 React + Vite 开发服务器配置。
- `backend/src/app.js`：说明 Express 服务、跨域配置、路由注册和统一错误处理。

## 2. 项目设计

- `contracts/GradeManager.sol`：说明链上成绩结构、管理员权限、学号哈希索引和事件设计。
- `backend/src/config/chain.js`：说明后端如何通过 ethers.js 连接本地链、创建管理员钱包和合约实例。
- `frontend/src/pages/Home.jsx`：说明前端页面分区、查询流程、管理员录入流程和系统架构展示。

## 3. 项目开发

- 智能合约开发：`contracts/GradeManager.sol`
- 服务层开发：`backend/src/services/contractService.js`、`backend/src/controllers/gradeController.js`
- 前端交互开发：`frontend/src/pages/Home.jsx`、`frontend/src/api/gradeApi.js`、`frontend/src/styles/app.css`

## 4. 部署与测试

- `scripts/deploy.js`：说明合约部署流程和部署产物生成。
- `test/GradeManager.test.js`：说明管理员权限、成绩录入、成绩修改、无记录查询和非法分数测试。
- `deployments/localhost/GradeManager.json`：部署脚本运行后生成，供后端读取合约地址和 ABI。

## 5. 报告中可引用的核心描述

本项目采用 Solidity 编写智能合约，使用 Hardhat 搭建本地以太坊测试链。合约部署者自动成为系统管理员，只有管理员可以录入或修改学生成绩，任意用户可以通过学号查询链上成绩。后端使用 Node.js、Express 和 ethers.js 对接本地链并封装 API，前端使用 React 实现成绩查询、管理员录入、交易状态展示和系统架构展示，形成完整的 DApp 开发流程。
