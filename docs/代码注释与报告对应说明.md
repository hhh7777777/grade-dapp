# 代码注释与报告对应说明

## 主要文件说明

- `contracts/GradeManager.sol`：定义成绩管理合约、管理员权限、成绩结构体、查询和写入方法。
- `scripts/compile.js`：使用 `solc` 编译合约，生成合约 artifact，并同步前端 ABI 文件。
- `scripts/deploy.js`：使用 `ethers.js` 连接 Ganache，把 `GradeManager` 部署到本地链。
- `deployments/ganache/GradeManager.json`：部署脚本运行后生成，供后端读取合约地址和 ABI。
- `backend/src/config/chain.js`：说明后端如何通过 `ethers.js` 连接 Ganache、创建管理员钱包和合约实例。
- `backend/src/config/contract.js`：读取部署文件，并校验合约地址和 ABI。
- `backend/src/services/contractService.js`：封装健康检查、成绩查询、成绩写入和交易状态查询。
- `frontend/src/pages/Home.jsx`：实现系统首页、成绩查询、管理员录入、MetaMask 连接和网络校验。
- `frontend/src/contracts/gradeManagerAbi.js`：前端调用合约所需 ABI，由编译脚本自动生成。
- `test/GradeManager.test.js`：基于 Ganache 的合约单元测试。

## 报告描述建议

本项目采用 Solidity 编写智能合约，使用 Ganache 搭建本地以太坊测试链。合约部署者自动成为系统管理员，只有管理员可以录入或修改学生成绩，任意用户可以通过学号查询链上成绩。后端使用 Node.js、Express 和 ethers.js 对接本地链并封装 API，前端使用 React 实现成绩查询、管理员录入、交易状态展示和系统架构展示，形成完整的 DApp 开发流程。
